1. Introduction

This document demonstrates the basics of writing in Quarto. Quarto is a powerful tool that allows you to combine standard Markdown text with executable code blocks (like Python, R, or Julia).

2. Markdown Formatting Options

Quarto uses standard Pandoc Markdown. Here are some of the most common formatting options available to you.

Text Formatting

You can format text to be bold, italic, or both. You can also use inline code to highlight variable names or commands. You can also strikethrough text.

Lists

Unordered List: * Apples * Bananas * Cherries * Dark Cherries * Rainier Cherries

Ordered List: 1. First step 2. Second step 3. Third step

3. Triggering Python Code in Quarto

To trigger Python code, you use standard markdown code blocks, but specify {python} at the beginning. Quarto will execute the code and embed the output directly into the final document.

You can control how the code behaves using special comments starting with #| (called “chunk options”).

Code 
# This is a simple Python code block
a = 15
b = 27
total = a + b

print(f"The sum of {a} and {b} is {total}.")
The sum of 15 and 27 is 42.

4. Graphing with Python Modules

Quarto seamlessly handles visual outputs from Python modules like matplotlib, seaborn, or plotly.

Below, we import numpy and matplotlib.pyplot to generate a graph. Notice the chunk options at the top: * #| label: gives the figure a cross-referenceable ID. * #| fig-cap: gives the figure a caption. * #| warning: false hides any system warnings from the final document.

Code 
# 1. Import our custom python file
import graph

# 2. Trigger the function
my_figure = graph.create_wave_plot()
